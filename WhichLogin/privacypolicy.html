WhichLogin Privacy Policy

Effective Date: 2-5-2026

Overview

WhichLogin is a browser extension that remembers which single sign-on (SSO) provider you use for specific websites to help you select the same provider on future visits.

Data Collection

WhichLogin does not collect, transmit, sell, or share personal data.

The extension does not collect:

  • Passwords
  • Authentication tokens
  • Cookies
  • Browsing history
  • Form data
  • Personal account credentials

Data Storage

WhichLogin stores a simple mapping between website domains and the SSO provider selected by the user (for example: example.com â†’ Google).

This data:

  • Is stored locally in the user's browser
  • Is not transmitted to external servers
  • Remains under the user's control

Data Sharing

WhichLogin does not share any data with third parties.

Remote Code

All extension functionality is packaged within the extension. No remote executable code is loaded or executed.

Changes

This policy may be updated if the extension's functionality changes.
